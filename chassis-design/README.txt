# Summary

This a robot chassis to build a line follower robot using the yellow gear motors and an arduino with a motor shield.

#### Parts needed:
  * 2x [yellow TT motor + wheel kit](https://www.banggood.com/Robot-Smart-Car-Wheel-Deceleration-DC-Motor-For-Arduino-Smart-Car-p-907973.html?p=QH1915587522201405O2)
  * 1x [Arduino Uno or clone](https://www.banggood.com/UNO-R3-ATmega328P-Development-Board-For-Arduino-No-Cable-p-964163.html?p=QH1915587522201405O2)
  * 1x [Motor shield for Uno](https://www.banggood.com/Motor-Drive-Shield-L293D-for-Arduino-Duemilanove-Mega-UNO-p-72855.html?p=QH1915587522201405O2)
  * 3x [Line sensor](https://www.banggood.com/5Pcs-5V-Infrared-Line-Tracking-Sensor-Module-For-Arduino-p-1089946.html?p=QH1915587522201405O2)
  * 1x [small LiPo battery](https://www.banggood.com/ZOP-Power-7_4V-400mAh-60C-2S-Lipo-Battery-JST-Plug-p-1173892.html?p=QH1915587522201405O2)
  * wires, standoffs, rubber band, nuts and screws

#### Notes
  * Instead of the three line sensor modules you can also use an [8 channel sensor array](https://www.banggood.com/Infrared-Detection-Tracking-Sensor-Module-8-Channel-Infrared-Detector-Board-For-Arduino-p-1194010.html?p=QH1915587522201405O2). The chassis has holes for this sensor board also. This array has analog output and you need an [Arduino Uno clone with additional A6 and A7 pins](https://www.banggood.com/RobotDyn-UNO-R3-ATmega328P-A6-A7-Pin-MicroUSB-Module-For-Arduino-DIY-p-1129191.html?p=QH1915587522201405O2) to use all 8 channels of the sensor
  * When you use LiPo packs without protection you should also use a [LiPo low voltage alarm](https://www.banggood.com/Wholesale-1S-8S-Li-po-Battery-Voltage-Tester-Checker-Indicator-Monitor-Buzzer-Alarm-p-26049.html?p=QH1915587522201405O2)
  * I use a screw in one of the front holes and let the front part of the robot slide on the screw.
